Welcome to Denios Linux

Security

The system is designed from the ground up to be secure. System is designed with privacy and security in mind, native support for full disk encryption, hardened debian core makes the system the perfect place to store sensitive data. No setup or Linux knowledge is required from your side its all been automated.

Easy to use

Denios is very simple and easy to use even for beginners.Denios is a live operating system that you can start on almost any computer from a DVD, USB stick, or SD card. Download the ISO file , burn it on a USB flash memory,CD,sd card etc,start the system. And that's it.

Freedom & Performance

The system is Free and Open Source and you can do whatever you want with it.. This is a lightweight system, so it can be run even on older machines. There are currently several themes with a single Desktop Environment, but you can install any and customize them all to your needs.

Home

This is designed for everyday use, privacy and software development. Security tools can be manually installed for putting together a custom and lightweight pentesting environment. Denios in both versions also supports much privacy, numerous anonymized and encryption security tools, and can be be run as Live-CD or from a USB Drive and installation mod.Both versions are free of tracking and telemetry "features" and provide many privacy tools such as AnonSurf, Tor Browser, custom Firefox profile, simple cryptographic tools and more.

Security

This OS that naturally preys on others is a security-centric free open-source Linux project for penetration testing and ethical hacking. You can also use it as a private, hardened, secure, and anonymized Linux distro.Denios Security Edition has around 1000 pre-installed tools which are split in several categories. Most kernel and user configs are customized by default to prevent any hacking, non-privileged access and reduce the attack surface. many built-in firewalls and defensive tools allow end-users to control the Denios.

Contribute

We welcome contributions to our project! If you are interested in getting involved, here are some ways you can help: Contact the project maintainers to introduce yourself and learn more about the project. Check out the open issues, package lists, If you see something you'd like to work on, let us know and we can assign it to you. If you have an idea for a new feature or improvement, please let us know by opening a new issue and marking it as a "feature request". Before you start working on a contribution, please make sure to read our contributing guidelines. These guidelines outline the process for submitting code, as well as the coding standards we follow. Thank you for considering contributing to our project! We look forward to working with you.
