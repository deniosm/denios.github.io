***************************************
Chapter 3 Anonsurf
***************************************

Anonsurf is one of the good anonymizing tools of Linux distribution. 
It helps us make our network tunnel secure. This tool uses TOR iptables to anonymize our network system.
You will need root access to use it. The default root password is: toor
